# Raspberry Pi Pico 资源极速精选

Pico是很好,不过其很多资源访问起来极其缓慢
Repo做了自动镜像,每天三次自动镜像到Gitee

## Repo功能大全

- 每天三次自动Mirror Pico官网资源到gitee
- 优化过的Pico开发环境建立脚本 pdusb-pico-setup.sh
- 每日自动编译最新example (WIP)
